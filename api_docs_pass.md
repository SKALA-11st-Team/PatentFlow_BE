# PatentFlow BE API Docs Index

이 파일의 예전 상세 API 초안은 현재 구현과 맞지 않아 제거했다.

현재 기준은 아래 문서를 사용한다.

- `AGENTS.md`: 공유 도메인 규칙, 최신 FR/UI ID, enum 계약
- `docs/PatentFlow_FR_mapping.md`: 최신 FR catalog, UI mapping, legacy FR mapping
- `docs/prompt.md`: 현재 BE-FE API 연동 요약

주의:

- 임원 승인 workflow는 현재 시스템 범위에 포함하지 않는다.
- `REPORT_GENERATED`는 현재 workflow 상태로 사용하지 않는다.
- 현재 AI 평가축은 `RIGHTS`, `TECHNOLOGY`, `MARKET`, `BUSINESS_ALIGNMENT` 네 가지다.
